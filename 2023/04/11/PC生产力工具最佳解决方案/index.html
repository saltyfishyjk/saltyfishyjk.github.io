PC生产力工具最佳解决方案

Part 0 前言

工欲善其事，必先利其器。这里记录个人的PC生产力工具最佳解决方案。如果没有特殊说明，均指Windows（10）平台下的软件。

Part 1 代码相关

Visual Studio Code

强大的编辑器，启动迅速，拥有丰富的插件生态。

GitHub

神。

GitHub Desktop

GitHub Desktop + 代理 = 最佳解决方案，完美解决不定时出现的git bash中pull和push失败的情况（笔者感觉可能是没有使用默认的网络通道，而是真正走了代理，所以速度很快），薄纱修改hosts等大众方法。

NewBing

不用美国手机号，太好了

适合读文献（edge打开PDF然后用bing侧栏）和检索。

ChatGPT

真香。

Clash

优秀的代理软件，不过偶尔可能会出现7890端口被占用的情况，重启电脑即可。

IDEA, PyCharm, CLion

分别是 Java, Python, C/C++ 的优秀 IDE，使用起来非常舒服。

Anaconda

最佳 Python 环境管理器。

Jupyter Notebook

优秀的记事本工具，适合将文档和代码块一起保存，最大优点在于代码可运行。

Part 2 非代码相关

PotPlayer

强大、快速、无广告的视频播放器。

Typora

最强大的markdown编辑器与解决方案。尽管需要付费，但是考虑到极高的使用频次以及一个激活码可以用三台设备可以拼车，还是很超值的。

知云文献翻译

强大的外文文献翻译工具。

EmEditor

强大快速的文本编辑器，是取代记事本和notepad++的最佳解决方案。

iSlide

PowerPoint导出高清图片的最佳解决方案。

draw.io

画图方便。

360zip

360少有的良心软件，最好使用国际版，强大快速。

Xshell

Windows下强大的终端软件，适合连服务器。（而且有教育版可以白嫖:+1:）

Xftp

一般和Xshell配套使用，带有图形化界面，可以本机和远程服务器文件互传，非常的方便:+1:

Adobe Acrobat DC

强大的PDF浏览和编辑软件。
