Marp笔记

Part 0 前言

Marp 是一个支持使用 markdown 快速制作精简大方的 PPT 的工具，在一些场景十分有用（特别是已经有了 markdown 文档且需要制作 PPT 汇报时）。目前的使用方式主要是在 VSCode 中安装 Marp 插件，进行类似经典 markdown 编辑器的双屏浏览（左边源码，右边实时渲染）。

优点：

  • 支持 markdown 语法
  • 不需要特别设置字体字号，不需要排版
  • 制作快速

  • 用Markdown写PPT ，Marp的果壳主题
